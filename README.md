# Immuny: Emergency UX Redesign Prototype

An interactive mobile app prototype built by Kappa Theta Pi (USC), showcasing a UX redesign of Immuny's allergy emergency app. Designed to minimize cognitive load and ensure users can navigate critical features with minimal thinking time during high-stress allergic reactions.

**Live:** [immuny.vercel.app](https://immuny.vercel.app)
**Client:** [immuny.ai](https://www.immuny.ai)

## Prototype Features

- Full iPhone mockup with touch-interactive screens
- Emergency response flow (detect reaction, call contacts, find ER)
- Medical ID screen (allergens, medications, emergency contacts)
- Find ER screen with map view
- Scanner screen for allergen detection
- Low-cognitive-load UI with clear visual hierarchy

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + inline styles
- **Deployment**: Vercel (static export)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design Principles

- Speed over features: critical actions reachable in one tap
- High contrast, large touch targets for emergency use
- Teal/green brand (#00C896) with dark backgrounds for accessibility
- Smooth transitions to reduce cognitive friction
